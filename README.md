# StudyConcepts Keyword Finder

StudyConcepts Keyword Finder is a simple web-based tool that helps identify low-competition, educational keyword opportunities for the StudyConcepts.in website.

It is designed specifically for concept-based, beginner-friendly educational content in fields such as Physics, Biology, Environment, Health, Astronomy, and General Knowledge.

The tool uses Google Trends data, keyword expansion, intent filtering, and SERP analysis to recommend keywords that have both demand and a high chance of ranking.

## Features

- Upload Google Trends CSV as input
- Automatically expand into long-tail keywords
- Filter for educational / informational intent
- Estimate competition using SERP analysis
- Categorize keywords by subject
- Rank keyword opportunities with a scoring system
- Export final keyword list as CSV
